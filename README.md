# python_input_create_pdf
    檔案需額外下載Releases底下big_file內的檔案，參照說明放置檔案位置🐷

##### 應用為產品標籤自動生成，格式為固定的每頁 15 格
##### 透過輸入產品名稱、製造與有效日期，以及需求的標籤張數，來自動產生對應數量的PDF檔案
